» Welcome to VOLUSIA.

First, start by selecting a proper XML configuration file of your NetInstall configuration. Export your complete configuration by right-clicking and exporting to XML on ORG level in the NetInstall Manager.

The default name for the XML file ist export.xml. If the file is present in the directory where Volusia resides it will be auto-loaded. Of course, you can specify another file or directory name. Your last choice will be remembered and auto-loaded if you start the tool again.

If the XML file is a valid NetInstall XML configuration file, your servers will show up in the selection box left to this message. You can select either one or more servers on which the chosen command will run. A brief description of the server will be displayed beneath the selection box.
Commands feature ping, service control, filecopy and WMI information. The respective output of the commands will be shown in this window.
