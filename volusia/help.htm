Getting started with Volusia
Select the XML file which contains your exported NetInstall Configuration. Usually this file goes by the name of "export.xml". Use the button to the right of the inputfield to select the directory.

NetInstallServers: select the servers on which the command will be executed. You can select with control-click or shift-click.

Command:
  • Ping:
    Sends a ping to all specifed servers. Results are shown in the output window.
  • Manage Services:
    Query, start or stop the three relevant NetInstall services on the specified servers. Select the local directory with the button. The target directory will be automatically adjusted.
  • Copy Files:
    Copy files from the local computer to the given directory on the specified servers. Mirroring Copy uses /MIR switch and results in identical directories, i.e. extra files on the target system will be deleted.
    MultiThread Copying runs simultaneously and not queued.
  • WMI Information
    Retrieves several informations from the specified servers, like shares, event log, processes etc. Hint: The command can take some minutes to execute.
